# MikeTalksTech

## Setup
1. Upload all files to your `miketalkstech.github.io` repo
2. Add `logo.png` and `photo.jpg` to the `images/` folder
3. Replace `YOUR_FORM_ID` in `contact.html` with your Formspree form ID
4. Optionally set custom domain in Settings > Pages

## Files
- `index.html` - Home
- `about.html` - About + bio
- `classes.html` - Class topics
- `contact.html` - Contact form
- `style.css` - All styles
- `images/` - Logo and photo

## Design
- Sora (headings) + DM Sans (body)
- #0C314B dark blue + #3B82F6 electric blue
- 800px max width
- Plain HTML/CSS, no dependencies
